# British Airways Excel Analysis

Excel-based analysis project focused on airline review patterns and business insights.

## Project Goal
Use structured review data to identify customer experience trends and support actionable service-improvement decisions.

## Business Questions
- What themes are most associated with negative reviews?
- Which dimensions of customer experience show strongest variation?
- How can insights be packaged for decision-makers in a concise format?

## Project Assets
- Final workbook: `deliverables/ba_airline_reviews_analysis.xlsx`
- Final presentation: `deliverables/british_airways_insights_presentation.pptx`
- Raw workbook snapshot: `data/ba_airline_reviews_raw.xlsx`
- Visual preview: `assets/presentation_preview.png`

## Repository Structure
- `data/`: source workbook artifact for analysis context.
- `assets/`: project visual preview.
- `deliverables/`: canonical final workbook and presentation.
- `docs/`: supporting notes and project guideline artifact.
- `notebooks_or_analysis/`: reproducibility notes.

## How To Review
1. Open `assets/presentation_preview.png` for a quick visual check.
2. Review `deliverables/british_airways_insights_presentation.pptx` for business narrative.
3. Open `deliverables/ba_airline_reviews_analysis.xlsx` to inspect transformation and analysis outputs.

## Cleanup Policy Applied
Legacy duplicates (e.g., multiple `final final` variants) were not published. This repo keeps one canonical workbook and one canonical presentation.
